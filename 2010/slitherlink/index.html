[Dingoo] Slitherlink

Slitherlink is a logic puzzle game created in 1989 by Japanese publisher Nikoli, famous for creating Sudoku, among many other puzzle games. See https://en.wikipedia.org/wiki/Slitherlink for more info.

I discovered the game a few years ago on the Nintendo DS with ‘Puzzle Series Vol. 5: Slitherlink’, which was never released outside of Japan. It’s such a good game I grew addicted to it, just like with Picross and Nurikabe.

This is my version of the game for Dingux (Dingoo Linux). It features 456 levels in four sizes: 5x5 (easy), 7x7, 10x10, 20x20 (hard). There’s also a save state function, undo button, and more.
